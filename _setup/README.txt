List of softwares for Windows:
------------------------------

- Unity 2021 LTS
	- Create an account
	- Download & install the Unity Hub software from unity.com/download
	- Through it, install the latest Unity 2021 LTS version, don't install any unecessary component from it.

- Visual Studio Code
	- C# extension
		- for writing code, 
		- autocompletion,

	    	>> https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp 

	- For the C# extension to work
		- .NET SDK 6.0 (lts)
		- Visual Studio Build Tools (for Windows)

		>> As stated in: https://code.visualstudio.com/docs/other/unity

	- Unity Debugger extension
		- "Debugger for Unity" by deitry
		- for debugging

	    	>> https://marketplace.visualstudio.com/items?itemName=deitry.unity-debug 

- Git for Windows + GUI client Tortoise Git
